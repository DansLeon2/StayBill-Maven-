# 🏨 StayBill-Maven
Sistema de gestión y facturación para empresas de hotelería desarrollado en **Java**.

## 🚀 Características
* **Gestión de Habitaciones:** Control de disponibilidad y estados.
* **Interfaz Moderna:** Diseño basado en **FlatLaf** con temas personalizados en JSON.
* **Base de Datos:** Persistencia de datos con **MySQL Workbench**.

## 🛠️ Tecnologías
* **Lenguaje:** Java 17
* **Gestor de dependencias:** Maven
* **UI:** Swing + FlatLaf
* **Base de Datos:** MySQL

## 📂 Estructura del Proyecto
* `src/main/java`: Contiene toda la lógica y controladores.
* `src/main/resources`: Aquí guardo el logo, las imágenes y los archivos JSON de la interfaz.
